# Student Performance Prediction

## Description

Student Performance Prediction is a machine learning web application that predicts a student's academic performance based on input features such as study hours, attendance, previous marks, and assignment scores. The application uses a trained machine learning model and provides predictions through a simple Flask-based web interface.

## Features

* Predict student performance instantly
* User-friendly web interface
* Machine learning model integration
* Flask backend for handling requests
* Trained model saved using Pickle



## Technologies Used

* Python
* Flask
* Pandas
* NumPy
* Scikit-learn
* Pickle
* HTML/CSS

## Installation

1. Clone the repository

git clone https://github.com/srinithiofficial2006-boop/student-performance-prediction.git

2. Install dependencies

pip install flask pandas numpy scikit-learn

3. Train the model


python mlmodel.py


4. Run the application


python backend.py


5. Open your browser and visit:


http://127.0.0.1:5000/
